# Convertidor de Unidades Gratis

Aplicación web gratuita para **convertir unidades** de longitud, peso, temperatura, área y volumen.

## ✨ Características

- 5 categorías: **longitud, peso, temperatura, área y volumen**.
- Sistema **métrico e imperial**.
- Botón de **intercambio rápido**.
- Conversión en tiempo real.
- Interfaz limpia y responsive.

## 🚀 Uso

1. Selecciona la **categoría**.
2. Ingresa el valor.
3. Elige las unidades de origen y destino.
